# Asset Placeholder Structure

- `images/`: ship illustrations, skyboxes, island art, decorative UI textures.
- `ui/`: banners, icon atlases, health segment art, button skins.
- `audio/`: cannon, impact, sinking, click SFX and optional music loops.
- `effects/`: particles, sprite sheets, smoke, splash overlays.

The game currently uses CSS/canvas placeholders so these files can be swapped in later without changing game logic.
